# RAPA Analyzer: Chrome Extension & Data Analysis Tool 🎪📉

Профессиональный набор инструментов для анализа данных соревнований Федерации воздушной гимнастики России (RAPA).

## 🚀 Основные возможности

### 🧩 Расширение для Chrome (RAPA Analyzer)
- **Мгновенный сбор данных:** Загрузка участников с сайта `referee.f-rapa.ru` в один клик.
- **Интерактивный дашборд:** Отдельная вкладка с таблицами и продвинутой фильтрацией.
- **Умные фильтры:** Фильтрация по городам, школам, дисциплинам и категориям с автообновлением.
- **Продвинутая статистика:** Автоматический расчет возраста, подсчет уникальных участников и статистика по школам.
- **Интеграция с Google Sheets:** Прямой экспорт и импорт данных из облака через OAuth2.
- **Надежный экспорт в Excel:** Создание структурированных XLSX файлов с автофильтрами.

### 🖥 Python инструменты для анализа
- **Веб-приложение Streamlit:** Удобный интерфейс для анализа локальных Excel файлов (`app.py`).
- **CLI скрипт:** Быстрый консольный генератор итоговых отчетов (`parse_excel.py`).
- **Нормализация школ:** Интеллектуальное объединение различных написаний названий школ.

## 📦 Установка и запуск

### Расширение для Chrome
1. Скачайте или клонируйте этот репозиторий.
2. Откройте Chrome и перейдите на страницу `chrome://extensions/`.
3. Включите **"Режим разработчика"** (вверху справа).
4. Нажмите **"Загрузить распакованное расширение"** и выберите папку `Chrome_Extension`.
5. (Опционально) Настройте интеграцию с Google Sheets по инструкции `Chrome_Extension/ИНСТРУКЦИЯ_GOOGLE.md`.

### Python инструменты
1. Убедитесь, что у вас установлен Python 3.10+.
2. Установите необходимые библиотеки:
   ```bash
   pip install streamlit pandas openpyxl requests
   ```
3. Для запуска веб-интерфейса используйте команду:
   ```bash
   streamlit run app.py
   ```
   Или запустите файл `Запуск_Анализа.command` (для пользователей macOS).

## 🛠 Структура проекта
- `Chrome_Extension/`: Все файлы расширения для браузера.
- `app.py`: Основное приложение Streamlit.
- `parse_excel.py`: Логика парсинга Excel и нормализации данных.
- `run_app.sh`: Вспомогательный скрипт запуска для macOS.
- `Запуск_Анализа.command`: Исполняемый файл для удобного запуска на Mac.
